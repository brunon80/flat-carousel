# Exemplo Carrossel animado feito em React Native e Flatlist

## Features
- Usando api nativa da animação
- Animated.Flatlist
- Roda na UI thread
- Somente React e React Native como dependencias
- Eventos disparados apartir da quantidade de movimento do scroll
- Typescript
